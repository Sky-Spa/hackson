## 意見共有

### 共通の認識

- 偶然性が欲しい
- テーマをオリンピックに絞る必要はなさそうだが今回はオリンピックでやる

### 競技のルールわからない問題

競技多すぎだし、オリンピックではじめて見る競技ばかりでルールがわからず  
面白味を理解するのが難しい。Wikipedia などにとばすなどして対応する。  

### オリンピック当事者の生配信

当事者が何かやる。YouTuberっぽく  
オリンピアンの食堂レポートがおもしろかったという印象がある。

### 自分以外の目線が重要

他の人から見た目線をアプリで共有できると良いのでは  
例：競技者目線を共有する。

### オリンピアンの本音

競技中に思っていたことを共有  
実はあの時、厳しかった/危なかったなど。
